# 1. OpenGL 설치하기
[OpenGL](https://www.opengl.org)은 그래픽스 관련된 작업을 하드웨어와 통신하기 위해서 만든 specification, 즉 소프트웨워와 하드웨어가 어떻게 통신할지 정해놓은 규격 같은 것이다. [Khronos Group](https://www.khronos.org)에서 관리하지만 실제 구현은 각 하드웨어 제조사가 맡고있다. 일반 PC라면 Intel, Nvidia, AMD등에서 하고 있을테고, 내가 쓰고있는 맥에서는 Apple이 당담했을 것이다. 사실 Apple은 이미 OpenGL을 버리고 Metal으로 넘어갔고, 업데이트는 끊긴지 오래다. 그래도 공부용으로는 충분하기 때문에 많은 강의에서 OpenGL 3.3을 이용해서 강의를 진행하고, LearnOpenGL도 이 버전을 기준으로 진행하고 있다.
## 1-1. 라이브러리 설치하기
내가 처음 프로그래밍을 배우기 시작했을 때 가장 궁금했던 것이 `#include <stdio.h>`었다. 나중에 가서야 standard IO, 즉 입출력을 당담하는 함수들을 미리 구현해놓고 모아놓은 header파일이었다는 것을 알게 되었다. OpenGL도 마찬가지로 이러한 것이 필요하고, LearnOpenGL에서는 GLFW와 GLAD를 이용한다.
### GLFW
* [GLFW](https://www.glfw.org/download.html)는 OpenGL이 창에 그림을 그리거나, 사용자의 입력을 도와주는 라이브러리이다. 다운로드 받으려고 페이지에 들어갔더니만, `Source package`, `GitHub repository`, `Windows binaries`, `MacOS binaries` 버튼이 있었다. 코스에 따라서 `Source package`를 다운로드 받고, 그 다음을 진행하려고 했지만... CMake니 Compilation이니 처음 보는 내용과 Windows에 Visual Studio (Code아님)을 기준으로 설명하고 있었기에 따라가기도 어려웠다. 학교에서는 가르쳐주지도 않았는데...
  

* 대신 [vcpkg](https://github.com/microsoft/vcpkg)라는 마이크로소프트의 툴을 이용해서 설치하기로 했다. vcpkg는 여러 라이브러리를 사용자의 환경에 맞도록 알아서 설치하도록 도와주는 프로그램이다. git을 쓸줄 모르므로 그냥 `Code`에서 `Download ZIP`을 통해서 다운로드 받았다. 그러면 `vcpkg-master`라는 폴더가 생기는데, 이제부터 여기서 터미널을 이용해서 작업을 진행할 것이다.
> 터미널은 윈도우의 파일 탐색기, 맥의 파인더의 역할을 그래픽 대신 글자로 한다. 간단한 명령어로썬
> > `cd (디렉토리(폴더))` : 해당 디렉토리로 이동  
> > `ls` : 현재 디렉토리의 내용을 표시  
> > `./(파일명)` : 파일 실행  
>
> 등이 있으며 처음 실행했을 때에는 보통 사용자의 루트 디렉토리에서 실행된다.

`Command + Space`를 이용해 Spotlight 검색에 `terminal`을 입력하면 터미널을 열 수 있다. 우리는 다운로드 디렉토리(폴더)의 `vcpkg-master`에서 작업을 할 것이니 `cd`라는 명령어를 이용해서 해당 디렉토리로 이동한다.
```terminal
cd Downloads/
cd vcpkg-master/
```
이제 우리는 (사용자이름)/Downloads/vcpkg-master에서 터미널을 이용하는 것이다. 굳이 위 명령어를 이용하지 않아도 finder를 통해 들어간 다음에 finder창 맨 아래의 directory에 적힌 vcpkg-master를 우클릭하여 `터미널에서 열기`를 눌러서 들어가도 된다. 이후 다운로드 받은 vcpkg를 설치하기 위해서 `bootstrap-vcpkg.sh`을 실행시킨다.
```
./bootstrap-vcpkg.sh
```
설치가 완료되면 `vcpkg-master`디렉토리에 vcpkg라는 실행파일이 생긴 것을 확인할 수 있다. 우리는 이제 이를통해 GLFW를 설치할 것이다. 터미널에서 계속해서 다음을 입력한다.
```
./vcpkg install glfw3:arm64-osx-dynamic
```
`install`, `glfw`, `osx`... 대충 단어를 보아하니 맥(osx) 버전의 glfw 3를 설치하는 것 같다. 이제 우리는 GLFW를 m1맥에 맞게 설치를 하였다.


### GLAD
GLAD는 운영체제마다 이름이 다르게 정해진 OpenGL 함수들을 쉽게 사용할 수 있게 해주는 라이브러리이다. LearnOpenGL에도 나와있는 과정도 쉽기 때문에 내용 그대로 [GLAD](https://glad.dav1d.de)에 들어가서 `3.3`과 `Core`를 선택하고 `GENERATE`를 눌러서 나오는 페이지에서 모두 다운로드 받으면 된다.
